🧠 Part 1 — PHP OOP (3 Hours)
📚 Topic 1: Inheritance

একটা class এর feature আরেকটা class reuse করা.

class User
{
    public $name = "Nirjhor";

    public function getName()
    {
        return $this->name;
    }
}

class Admin extends User
{
}

Use:

$admin = new Admin();
echo $admin->getName();

✅ Admin এখন User এর সব property + method use করতে পারবে.

🎯 Why Important For Laravel

Laravel এ inheritance everywhere:

Controller extends BaseController
Model extends Eloquent
Mailable extends Mailable
Command extends Command

মানে Laravel বুঝতে inheritance must.

📚 Topic 2: Method Overriding

Child class parent method change করতে পারে.

class User
{
    public function role()
    {
        return "User";
    }
}

class Admin extends User
{
    public function role()
    {
        return "Admin";
    }
}
📚 Topic 3: Parent Keyword

Parent method reuse + extend.

class User
{
    public function greet()
    {
        return "Hello";
    }
}

class Admin extends User
{
    public function greet()
    {
        return parent::greet() . " Admin";
    }
}
📚 Topic 4: Polymorphism Intro

Same method → different behavior.

class Payment
{
    public function pay()
    {
        return "Base payment";
    }
}

class BkashPayment extends Payment
{
    public function pay()
    {
        return "Paid via Bkash";
    }
}

class CardPayment extends Payment
{
    public function pay()
    {
        return "Paid via Card";
    }
}

Use:

$payments = [
    new BkashPayment(),
    new CardPayment()
];

foreach ($payments as $payment) {
    echo $payment->pay();
}

🔥 এটা later Strategy Pattern এর foundation.
