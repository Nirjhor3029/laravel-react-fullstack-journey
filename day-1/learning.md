📚 Topic 1: Class & Object

Real world entity কে code এ represent করার way.

class User
{
    public $name;

    public function setName($name)
    {
        $this->name = $name;
    }
}

$user = new User();
$user->setName('Nirjhor');
✅ Must Understand
class = blueprint
object = real instance
new = object create
$this = current object reference
