🧠 Part 1 — PHP Traits (1h)
📘 Definition (Interview Ready)

Traits are a mechanism for code reuse in multiple classes without inheritance.

বাংলায় সহজ:

Multiple class এ same method use করতে চাও কিন্তু inheritance chain এর problem আছে → traits use করো।

💻 Example
trait Logger {
    public function log($message) {
        echo "Log: $message";
    }
}

class User {
    use Logger;
}

class Admin {
    use Logger;
}

$user = new User();
$user->log("User logged in");

$admin = new Admin();
$admin->log("Admin logged in");

✅ output:

Log: User logged in
Log: Admin logged in

Interview Tip:

“Traits allow horizontal code reuse across classes, not just vertical like inheritance.”

🧠 Part 2 — Static Methods & Properties (45min)
📘 Definition

Static methods/properties belong to the class itself, not instances.

বাংলায়:

Object create না করেই access করতে পারি।

💻 Example
class Config {
    public static $appName = "FullstackApp";

    public static function getAppName() {
        return self::$appName;
    }
}

// Access without creating object
echo Config::$appName;
echo Config::getAppName();

Interview Tip:

“Static used for shared data, constants, or utility functions.”

🧠 Part 3 — Magic Methods (45min)
📘 Definition

Magic methods are predefined methods in PHP starting with __ that perform special tasks.

Examples:
__construct() → object creation setup
__destruct() → cleanup
__get($property) → property read
__set($property, $value) → property write
__call($method, $args) → call undefined method
💻 Example: __get & __set
class User {
    private $data = [];

    public function __set($name, $value) {
        $this->data[$name] = $value;
    }

    public function __get($name) {
        return $this->data[$name] ?? null;
    }
}

$user = new User();
$user->name = "Nirjhor";
echo $user->name; // Nirjhor

⚛️ Part 4 — React Event Handling (1h)
📘 Definition

Event handling lets you respond to user actions like clicks, form submissions, typing.

💻 Example: Button Click
import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  function handleClick() {
    setCount(count + 1);
  }

  return <button onClick={handleClick}>Count: {count}</button>;
}
💻 Example: Input Change
import { useState } from "react";

function NameInput() {
  const [name, setName] = useState("");

  return (
    <input
      value={name}
      onChange={(e) => setName(e.target.value)}
      placeholder="Enter name"
    />
  );
}

Interview Tip:

“React synthetic events normalize browser events and handle updates efficiently.”
